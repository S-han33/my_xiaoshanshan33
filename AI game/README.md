# AI知识答题小游戏

## 项目简介
这是一个帮助AI小白加深对AI认识的答题小游戏，内容涵盖AI基础知识、大语言模型、提示工程、AIGC等前沿话题。
网页游览：https://s-han33.github.io/my_xiaoshanshan33/AI%20game/index.html
## 知识库说明

### 题库结构
- **题库文件**: `questions.json`
- **总题目数**: 80道
- **难度分级**: 3级（1=入门，2=进阶，3=前沿）

### 知识分类（7大类）
1. **AI基础概念** - 人工智能的基本术语和历史
2. **大语言模型** - LLM的核心知识和原理
3. **提示工程** - 与AI对话的技巧和方法
4. **AIGC与生成式AI** - AI内容生成技术
5. **模型训练与优化** - AI模型是如何训练出来的
6. **AI应用与工具** - 当前主流AI产品和应用
7. **前沿技术** - AI领域最新技术和趋势

### 难度说明
- **入门（难度1）**: 基础术语和概念，适合零基础用户
- **进阶（难度2）**: 理解原理和应用，适合有一定了解的用户
- **前沿（难度3）**: 最新技术和趋势，适合想深入了解的用户

## 题目格式
每道题目包含：
- `id`: 题目编号
- `category`: 所属分类
- `difficulty`: 难度等级（1-3）
- `question`: 题目内容
- `options`: 4个选项
- `answer`: 正确答案索引（0-3）
- `explanation`: 答案解析
